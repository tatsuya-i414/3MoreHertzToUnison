【ループクエイク（振動）プラグイン】

■使い方
このテキストが入っているフォルダごと「data/others/plugin」フォルダに置きます。
その後、first.ksとかに以下のように記述してください。

[plugin name=loop_quake]
指定可能属性：なし

記述した時点から後述のタグを使用可能になります。


■できること
・レイヤーを指定して振動
・縦揺れと横揺れを同時指定
・停止タグまで永続して振動
※全レイヤーを一括で振動させることは（いまのところ）できないので、デフォルトの[quake]タグと併用推奨です。


■タグと指定可能属性
loop_quake  ：振動開始
属性：
layer   ：振動させるレイヤーを指定します。「0,1,2」のようにカンマ区切りで複数指定可能です。未指定の場合、"base"となります。
vmax    ：縦の揺れ幅を指定します。hmaxと同時指定可能です。単位はピクセルです。
hmax    ：横の揺れ幅を指定します。vmaxと同時指定可能です。単位はピクセルです。
time    ：1回あたりの揺れにかかる時間を指定します。単位はミリ秒です。未指定の場合、100ミリ秒となります。
effect  ：揺れのイージング効果を指定します。[anim]タグのeffectパラメータと同じものが指定できます。未指定の場合、 "linear"となります。

loop_quake_stop ：振動停止
属性：
layer   ：振動停止させるレイヤーを指定します。「0,1,2」のようにカンマ区切りで複数指定可能です。未指定の場合、"base"となります。
